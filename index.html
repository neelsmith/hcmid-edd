Scholarly editions from the Holy Cross Manuscripts, Inscriptions and Documents Club

This is a repository for notes about the scholarly editions being developed by the Holy Cross Manuscripts, Inscriptions and Documents Club.

All archival work is managed using git repositories. Here are links to repositories on github for club projects in the fall of 2013:

  • Geneva Lat.MS. 49, Jerome's Chronicles
  • St. Gallen, Vadianische Sammlung 298, Jerome's Chronicles
  • Primary sources for teaching introductory Latin
  • Texts of Archimedes
  • Classical Athenian epigraphic documents related to tribute
  • Codex Bodmer 48, Chaucer's Canterbury Tales
  • Manuscripts with musical notation using neumes
  • As part of the Homer Multitext project
    • Iliad 10 in the Venetus A and Escorial Upsilon 1.1 MSS
    • Iliad 22 in the Venetus A and Escorial Upsilon 1.1 MSS
